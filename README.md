# E-commerce Website

A minimal static e-commerce website prototype demonstrating a simple storefront layout, product links, and responsive styling.

**Overview**

- Small static site built with HTML and CSS for learning and prototyping.

**Files**

- `index.html` — Main storefront page.
- `link.html` — Example product / link page.
- `link.css` — Stylesheet for layout and responsive styles.

**Quick Start**

1. Open `index.html` in your browser (double-click or drag into a browser window).
2. (Optional) Run a simple HTTP server for local testing:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

**Development**

- Edit the HTML files and `link.css` directly; refresh the browser to see changes.
